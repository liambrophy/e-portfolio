  • MovieSearch Project

    HTML, CSS, JavaScript, React

    Website that allows users to search for thousands of movies by their title. The user can then click on each movie displayed for further information on that movie. All search and movie data is pulled from a real world API. Built using React, JavaScript, HTML and CSS best practices.

  • Library E-Commerce Project

    HTML, CSS JavaScript, React

    An e-commerce website where users can shop for a range of books. The data is pulled from a mock API to simulate a real world e-commerce website. Books can be sorted by price and rating. Implemented a checkout feature that allows users to view the items in their basket as well as the total price. Built using React, JavaScript, CSS and HTML best practices.

  • Netflix Clone Project

    HTML, CSS JavaScript, React, TypeScript

    A complete clone of Netflix. Sign-in and sign-up functionality added by using Firebase. Users can add movies or shows to 'My List' which is kept track of by Firebase. Subscription functionality implemented using Stripe. Users can select from multiple plans, and also cancel or change these plans at any time. Made use of multiple custom hooks. Tailwind CSS was used for improved styling. All data is pulled from a real world API to ensure movies and shows are kept up to date.
